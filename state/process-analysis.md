## ⚠️ Проблемы процесса

### 1. Staging окружение не работает

**Проблема:** Staging бот запущен (port 8081), но Telegram требует порт 443 для вебхука.

**Причины:**
- Нет nginx reverse proxy
- Нет ngrok туннеля
- Нет публичного HTTPS URL

**Влияние:** Невозможно провести интеграционные тесты на staging

**Решение (варианты):**
- Установить nginx (8081 → 443)
- Использовать ngrok для временного туннеля
- Пропустить staging для простых фиксов
