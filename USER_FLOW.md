# Agon Arena — пользовательский сценарий

1. Пользователь открывает Telegram-бота и нажимает `/start`.
2. Бот показывает главное меню с кнопками:
   - начать поединок
   - отправить реплику
   - следующий раунд
   - завершить поединок
   - результаты
   - сценарии
   - правила
3. Пользователь стартует поединок.
4. Бот создаёт `Duel`, 2 `DuelRound`, подставляет сценарий и показывает стартовую реплику AI для раунда 1.
5. Пользователь отправляет текстовые ходы; каждый ход сохраняется в `duel_messages`, AI отвечает в роли оппонента.
6. Пользователь нажимает `Следующий раунд`.
7. Бот завершает раунд 1, меняет роли, запускает раунд 2 и показывает стартовую реплику AI для второго раунда.
8. Пользователь снова делает текстовые ходы во втором раунде.
9. Пользователь нажимает `Завершить поединок`.
10. Бот собирает стенограммы двух раундов, прогоняет их через трёх судей (`owner`, `team`, `sending_to_negotiation`) и сохраняет `JudgeResult`.
11. Бот показывает итоговый вердикт и позволяет посмотреть результаты последнего поединка.
