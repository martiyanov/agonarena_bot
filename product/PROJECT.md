# Agon Arena — PROJECT

## Что это

Agon Arena — Telegram-бот для практики управленческих поединков.

Текущий MVP — это быстрый текстовый duel-flow:
- человек против AI;
- 2 раунда;
- во 2-м раунде меняются роли;
- после поединка 3 AI-судьи дают итоговый разбор.

## Ценность продукта

Бот даёт безопасную среду, где можно:
- тренировать жёсткие переговоры;
- отрабатывать управленческие конфликты;
- сравнивать подходы в двух раундах;
- получать короткий разбор с трёх точек зрения.

## Для кого

- руководители;
- тимлиды;
- предприниматели;
- специалисты, которым нужно тренировать переговоры и конфликтные разговоры.

## Текущий MVP

Входит в MVP:
- Telegram-бот;
- 10 готовых сценариев из seeds;
- выбор готового сценария из списка;
- случайный сценарий;
- пользовательский сценарий из свободного описания;
- 2-раундовый поединок с AI-оппонентом;
- сохранение сообщений и результатов в SQLite;
- 3 виртуальных судьи;
- текстовые и голосовые пользовательские реплики;
- кнопочный UX.

Не входит в MVP:
- веб-интерфейс;
- multi-player режим;
- аналитика уровня кабинета/дашборда;
- пагинация и фильтры сценариев;
- админ-панель;
- сложная scoring-модель судей.

## Основные UX-принципы MVP

- Telegram — основной интерфейс.
- Основные действия должны быть понятны без чтения документации.
- Пользователь выбирает сценарий явно, а не вводит код вручную.
- Внутри поединка используется одна основная action-кнопка: **«Завершить раунд»**.

## Целевое состояние MVP

MVP считается приведённым в порядок, когда:
- выбор сценария в Telegram понятен и короток;
- duel-flow стабилен и покрыт тестами;
- runtime после deploy совпадает с workspace;
- ручная приёмка Telegram UX проходит без регрессов.
