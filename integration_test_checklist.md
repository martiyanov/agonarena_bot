# Чеклист интеграционного теста: Фикс кнопки "🎯 Сценарии"

## Цель
Проверить, что после фикса кнопка "🎯 Сценарии" в Telegram-боте АгонАрены корректно отображает список сценариев, а не сообщение "Поединок уже идёт...".

## Предусловия
- Бот запущен и работает (проверено по логам)
- Код в Docker-контейнере обновлён (функция `handle_start_button` изменена как нужно)

## Шаги проверки

1. **Открыть чат с ботом**
   - Отправить команду `/start`
   - Убедиться, что отображается главное меню

2. **Проверить кнопку "🎯 Сценарии"**
   - Нажать на кнопку "🎯 Сценарии"
   - Убедиться, что появляется список сценариев (а не сообщение "Поединок уже идёт...")

3. **Проверить функциональность сценариев**
   - Выбрать любой сценарий из списка
   - Убедиться, что сценарий запускается корректно

4. **Проверить повторный вход**
   - Выйти из сценария (например, через команду `/menu`)
   - Снова нажать "🎯 Сценарии"
   - Убедиться, что снова отображается список сценариев

5. **Проверить другие кнопки меню**
   - Убедиться, что другие кнопки главного меню работают как обычно

## Ожидаемый результат
- При нажатии на "🎯 Сценарии" всегда отображается список сценариев
- Не появляется сообщение "Поединок уже идёт..." при первом нажатии
- Все сценарии доступны для выбора и запуска
