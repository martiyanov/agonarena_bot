# Agon Arena — PROJECT

## Цели

- Создать Telegram-бота для **практики управленческих поединков** в формате экспресс-MVP.
- Дать пользователю безопасную «арену», где можно тренировать переговорные и управленческие навыки.
- Использовать AI как:
  - оппонента в поединке;
  - трёх виртуальных судей с разными перспективами.

## Целевая аудитория

- Руководители и тимлиды.
- Предприниматели и собственники.
- Люди, которым нужно тренировать «жёсткие» переговоры и управленческие решения.

## Формат поединка (MVP)

- Формат: **экспресс-поединок**.
- Участники: человек против AI.
- Раунды: 2 раунда.
  - Во втором раунде — **смена ролей** между игроком и AI.
- Судьи (3 виртуальные роли):
  - Собственник.
  - Команда.
  - Отправляющий на переговоры.
- Интерфейс: **button-first** (минимум свободного ввода, основные действия через кнопки).

## Роли в системе

- **Игрок (человек)** — запускает поединок, выбирает сценарий, пишет реплики.
- **AI-оппонент** — в каждом раунде отвечает в заданной роли.
- **3 AI-судьи** — после раундов дают оценку и короткие комментарии с разных позиций.
- (Позже) **Админ/редактор** — добавляет сценарии, настраивает параметры поединков.

## Границы MVP v1

Входит:
- только текстовые поединки;
- один режим: человек против AI;
- фиксированный набор сценариев поединков;
- 2 раунда + смена ролей;
- 3 виртуальных судьи с простыми, но осмысленными комментариями;
- хранение базовой статистики (кол-во поединков, сценарии, результаты).

Не входит в v1 (можно для v1.1+):
- несколько игроков-человек одновременно;
- веб-интерфейс;
- сложная аналитика и отчёты;
- интеграция с внешними CRM/таск-трекерами.
